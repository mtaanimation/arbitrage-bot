## What is a Crypto Arbitrage Bot?

A **Crypto Arbitrage Bot** is an automated trading tool that detects price discrepancies for the same crypto asset across different exchanges and exploits them for profit. The **Crypto Arbitrage Bot** performs buy-sell cycles faster than any human trader, allowing for passive, hands-free income opportunities.

## Installation Steps

1. **Download the Bot Source Code** (unzip if necessary).
2. Open **Command Prompt** in the project directory.
3. Run the following:

```bash
npm i .
```

This installs all required packages for the **Crypto Arbitrage Bot for Automated Profits**.

---

## How to Configure Your .env File

Inside your project folder, create a `.env` file with the following format:

```env
PRIVATE_KEY=your_private_key
RPC_URL=your_rpc_provider_url
DEX1=https://api.dex1.com
DEX2=https://api.dex2.com
THRESHOLD=0.5
```

This config powers the **Crypto Arbitrage Bot** to scan and act.

## Running the Bot on Windows

To launch the **Crypto Arbitrage Bot**, open your command line inside the folder and run:

```bash
node index.js
```

The bot will begin scanning price differences and executing profitable trades automatically.
